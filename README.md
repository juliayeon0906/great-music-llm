# Great Music - Classical Orchestra Website

A beautiful, elegant website for a classical orchestra built with React and Tailwind CSS v4.

## Features

- **Hero Section** - Stunning full-screen hero with search and navigation menu
- **About** - Orchestra introduction with elegant animations
- **Events** - Upcoming concert listings with booking functionality
- **Artists** - List of our musicians
- **Footer** - Contact information and quick links

## Tech Stack

- React 18
- Tailwind CSS v4
- Framer Motion (for animations)
- Vite
- Lucide React (icons)
